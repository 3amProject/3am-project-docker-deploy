# 🚚 새벽 세시

## 1. 서비스 소개(Introduction)

식단 배송 서비스 [새벽다섯시](https://www.5am.co.kr/shop/calendar_shop.php)를 클론코딩하여, ‘새벽 세시’ 웹 서비스를 구현하는 프로젝트

<br>

## 2. 사용 스택 & 개발 환경(Stack)

### Front-end

- JavaScript(ES6)
- React / React Router
- Styled Component

### Back-end

- Java
- Spring Boot
- MyBatis
- MySQL
- JSON Web Token

### Tools

- STS4
- VSCode
- IntelliJ
- Git / Github
- Notion, Slack

### OS

- Windows, MacOS

<br>

## 3. 주요 기능(Features)

- 회원가입 / 로그인
- 회원 정보 조회 및 수정
- 상품 리스트 조회
- 날짜, 메뉴 선택 후 장바구니 담기
- 장바구니에서 담긴 상품 조회, 수정, 삭제, 주문
- 회원 / 비회원 주문
- 주문정보 조회
